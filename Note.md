#### 项目目的
- 解耦和
- 一拆六
- 以前就一个manage文件
  - manage进行全局控制
  - 在应用初始化做初始化
    - 初始化配置
    - 初始化路由
    - 初始化第三方
  - 配置文件
    - 配置项目所需各种信息
  - 视图函数
    - 用来处理业务逻辑，协调模板和模型
  - 模型文件
    - 定义模型
  -外部扩展
    - 统一管理扩张

#### 项目结构
- manage  用于控制程序
- app/\_\_init__  初始化文件
  - 初始化整个Flask对象 ，以及Flask所有的各种插件
- setting  配置文件
  - 配置整个项目运行环境
- exts  项目扩展库
  - 第三方扩展打包处理
- views  视图函数
  - 处理业务逻辑
  - 协调模板和模型之间的关系
- models  模型
  - 定义模型结构
  - 获取数据库中表的关系映射


#### 数据库迁移
- flask db init
  - 初始化指令，只可调用一次
- flask db migrate
  - 生成迁移文件 migration/versions/*.py
- flask db upgrade
  - 执行迁移文件
  - 根据迁移文件进行升级
  - 数据库内容升级
  - 参数 --message 'msg' 对迁移添加日志
- flask db downgrade
  - 执行迁移文件
  - 数据库降级
